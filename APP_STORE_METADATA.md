## Description

Turn your camera roll into the ultimate party game! PhotoParty is a hilarious multiplayer game where friends compete to find the best photos from their libraries.

**HOW IT WORKS**

1. Create a game and share the code with friends
2. Everyone sees the same funny prompt - "Dying Laughing", "Questionable Life Choices", "Peak Chaos Energy"
3. Race to find the perfect photo from your camera roll
4. Vote on your favorite submission
5. The best photo wins the round!

**FEATURES**

- Play with 2-8 friends on the same WiFi
- 150+ hilarious prompts across 10 categories
- Quick games (3-15 rounds)
- Beautiful dark mode design
- Smooth animations and haptic feedback
- No accounts required - just play!

**PRIVACY FIRST**

Your photos never leave your local network. No uploads, no servers, no tracking. Photos are shared directly between devices and deleted when the game ends.

**PERFECT FOR**

- House parties
- Game nights
- Family gatherings
- Road trips
- Any time you want to laugh with friends!

Warning: May cause excessive laughter and embarrassing photo reveals!

## App Review Notes

PhotoParty is a local multiplayer party game that requires 2+ devices on the same WiFi network to play.

To test the app:
1. Install on two or more iOS devices
2. Open the app on all devices
3. On one device, tap "Create Game"
4. On other devices, tap "Join Game" and either enter the code or select the nearby game
5. The host can start the game once 2+ players have joined

The app uses MultipeerConnectivity for local networking and PHPickerViewController for photo selection.
